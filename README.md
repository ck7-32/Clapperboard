# 數位場記板 Clapperboard

一個純前端的網頁版電影場記板,可記錄拍攝設備、拍攝場地、日期,以及「場 / 鏡 / 次」(Scene / Shot / Take)編號。所有資料自動保存在瀏覽器本機 (localStorage),重新整理或關閉後不會遺失。

## 功能

- 製作資訊欄:拍攝設備、拍攝場地、日期
- 場 / 鏡 / 次 計數器,可手動輸入或用 ＋／− 快速調整
- 資料自動儲存於瀏覽器本機

## 本機預覽

直接用瀏覽器打開 `index.html` 即可,無需建置工具。

## 部署到 GitHub Pages

push 到 GitHub 後,在 repo 的 **Settings → Pages** 選擇 `main` 分支根目錄即可。網址為 `https://<使用者名稱>.github.io/<repo 名稱>/`。
